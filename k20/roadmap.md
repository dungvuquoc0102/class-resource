## Module 6: Javascript - Trái tim của Web

- Javascript căn bản
  - Javascript: Ngôn ngữ lập trình web
  - 3 cách tích hợp JS vào web
  - Biến: var, let, const và scope
  - Kiểu dữ liệu trong Javascript
  - Toán tử và biểu thức
  - If/else và switch case
  - Vòng lặp: for, while, do-while

- Làm việc với dữ liệu
  - String: Xử lý chuỗi trong thực tế
  - Array: Duyệt và thao tác mảng
  - Array methods: map, filter, reduce
  - Object: Khai báo, thêm, sửa, xóa
  - Object nâng cao: Nested objects và arrays
  - JSON: Parse và stringify
  - Number: Format số và tiền tệ
  - Date: Làm việc với date-fns
  - Thực hành: Xử lý data phức tạp

- DOM và Events

  - Query và thao tác với DOM
  - Events: Click, Submit, Keyboard, Mouse
  - Event Object: preventDefault, stopPropagation
  - Storage: LocalStorage và SessionStorage
  - Thực hành: Todo App với LocalStorage

- Javascript nâng cao
  - Try Catch: Xử lý lỗi
  - Throw Error tùy chỉnh
  - Promises: Then, catch, finally
  - Async/Await: Xử lý bất đồng bộ
  - HTTP/HTTPS: Request và Response
  - Fetch API: Gọi API thực tế
  - Thực hành: Weather App với API

- Modern Javascript (ES6+)
  - Destructuring: Object và Array
  - Spread và Rest operators
  - Arrow functions
  - ## Modules: Import và Export

- Tools và Build
  - NPM: Quản lý packages
  - Thư viện phổ biến: lodash, date-fns
  - SPA vs MPA: Kiến trúc web apps
  - Vite: Modern build tool
  - Thực hành: SPA với Vite
