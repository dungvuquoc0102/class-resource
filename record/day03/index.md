# Tổng quan về CSS và thực hành xây dựng giao diện HTML, CSS cơ bản

## 1. Tổng quan về CSS

- Định nghĩa: CSS là ngôn ngữ định kiểu cho các phần tử HTML.
- Cú pháp:

```css
selector {
  property: value;
}
```

- Selector: Bộ chọn xác định phần tử HTML nào cần áp dụng kiểu dáng.
  Phân loại selector:
  - Selector cơ bản: element, class, id.
  - Selector kết hợp: descendant, child, next, sibling.
